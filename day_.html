Pebcessing Reference

Processing-like framework for Pebble Smartwatch. Let's sketch a watchface with code!

day()

Examples

void draw() {
  fill(0);

  int d = day();    // Values from 1 - 31
  int m = month();  // Values from 1 - 12
  int y = year();   // 2003, 2004, 2005, etc.

  // C-style string
  char dayStr[3];
  char monthStr[3];
  char yearStr[5];
  snprintf(dayStr, 3, "%d", d);
  snprintf(monthStr, 3, "%d", m);
  snprintf(yearStr, 5, "%d", y);

  text(dayStr, 10, 28);
  text(monthStr, 10, 56); 
  text(yearStr, 10, 84);  
}

Description

The day() function returns the current day as a value from 1 - 31.

Syntax

day()

Returns

int
