Pebcessing Reference

Processing-like framework for Pebble Smartwatch. Let's sketch a watchface with code!

noLoop()

Examples

void setup() {
  noLoop();
}

void draw() {
  // draw something
}

// For use minuteEvent(), define ENABLE_MINUTE_EVENT in settings.h
void minuteEvent() {
  redraw();  // or loop()
}

Description

Stops Pebcessing from continuously executing the code within draw(). If loop() is called, the code in draw() begins to run continuously again. If using noLoop() in setup(), it should be the last line inside the block. When noLoop() is used, it's not possible to manipulate or access the screen inside event handling functions such as mousePressed() or keyPressed(). Instead, use those functions to call redraw() or loop(), which will run draw(), which can update the screen properly. This means that when noLoop() has been called, no drawing can happen.

Syntax

noLoop()

Returns

void
